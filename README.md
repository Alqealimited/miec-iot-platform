# MIEC IoT Platform

Open-source indoor environmental health monitoring platform for the MIEC (Modular Integrated Evaporative Cooling) system.

## Overview

MIEC is a solar-powered indoor climate system that monitors and improves indoor environmental conditions in off-grid households. This repository will contain the open-source components of the IoT monitoring layer.

## Planned Open-Source Components

- IoT sensor firmware (13 environmental parameters)
- GSM data transmission protocols
- Cloud dashboard code
- Environmental data schema
- USSD feature phone interface

## Status

This repository is under active development as part of the UNICEF Venture Fund Climate Ventures programme. Initial code release is planned for Q3 2026.
